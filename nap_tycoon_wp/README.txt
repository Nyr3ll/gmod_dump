# Addon GMod - Système Tycoon personnalisable

## Description
# NOTE : La banque à WaterPoints / WaterMoney n'est pas fonctionnelle.
Cet addon orienté **Tycoon** a été conçu pour fonctionner sur la **majorité des gamemodes**.  
Il introduit un système de **monnaie interne au serveur**, distinct de l'argent habituel, et permet aux joueurs d’acheter des entités pour **produire cette monnaie**, ainsi que d’autres objets comme des armes ou des entités spéciales (ex. : **"Combine Mine"**).

Tout se gère depuis un **menu unique** accessible via la commande **`!nap_tycoon`**.  
> 📁 La commande est modifiable dans le fichier de configuration : `lua/nap_tycoon_wp/main_config.lua`.

---

### ⚙️ Fonctionnalités principales :

- 💰 Création d’un **système de monnaie alternatif**, indépendant de l’économie du gamemode.
- 🛠️ Achat d'entités pour **générer de la monnaie** ou obtenir divers objets (armes, pièges...).
- 🌍 Système de **langue intégré** (Français et Anglais disponibles).
- 🧾 Toutes les **commandes utiles** sont documentées en commentaire dans le fichier `main_config.lua`.

---

## Description (English)
# NOTE : Bank of WaterPoints / WaterMoney is not functionnal.
This **Tycoon-style addon** is designed to be compatible with **most GMod gamemodes**.  
It introduces a **custom internal currency system** that allows players to buy entities which generate more of this currency, along with other items such as weapons or special entities like **"Combine Mine"**.

Everything is managed through a **single menu** accessible with the **`!nap_tycoon`** command.  
> 📁 The command can be changed in the config file: `lua/nap_tycoon_wp/main_config.lua`.

---

### ⚙️ Main Features:

- 💰 Create an **alternative in-server currency** system, independent from the main gamemode economy.
- 🛠️ Buy farming entities, weapons, and unique objects using this currency.
- 🌍 Built-in **language support** (French and English available).
- 🧾 All **useful commands** are commented in the `main_config.lua` file for easy access.

## 📁 Installation

1. Téléchargez ou clonez le dépôt.
2. Placez le dossier dans `garrysmod/addons/`.
3. Modifiez `lua/nap_tycoon_wp/main_config.lua` selon vos besoins :
   - Commande d’accès
   - Langue
   - Contenu débloquable
   - Etc.
