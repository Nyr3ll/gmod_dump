# Addon GMod - Menu to buy EXP on Basewars

## Description

Cet addon a été **spécialement développé pour le gamemode Basewars**.  
Il ajoute un **menu interactif** permettant aux joueurs d'acheter de l'expérience (XP) en échange d'argent en jeu.

### Fonctionnalités :
- 📥 Achat d'XP via la commande `**/buyxp**` (modifiable dans le fichier `bw_menu_achat_xp/client/vgui_menu_xp.lua`)
- ⚙️ Système de **sauvegarde des préférences client** :
  - Affichage ou masquage d'éléments du menu
  - Personnalisation de la **couleur de l'interface**
- 💾 Les paramètres sont sauvegardés **localement** pour chaque joueur

---

## Description (English)

This addon was **specifically developed for the Basewars gamemode**.  
It adds an **interactive menu** that allows players to buy experience (XP) using in-game money.

### Features:
- 📥 Buy XP using the `**/buyxp**` command (can be changed in `bw_menu_achat_xp/client/vgui_menu_xp.lua`)
- ⚙️ **Client-side preference saving** system:
  - Show/hide parts of the menu
  - Customize the **menu color**
- 💾 Settings are saved **locally per player**
