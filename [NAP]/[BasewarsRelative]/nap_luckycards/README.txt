# Addon GMod - Scrathing card

# NOTE Cet addon n'est pas terminé mais la 1er carte à gratter est fonctionnelle !
## Description

Cet addon pour Garry's Mod a été initialement conçu pour le mode de jeu **Basewars**, mais il est facilement adaptable à d'autres gamemodes tels que **DarkRP** ou tout autre système disposant d'une économie (argent) ou d'un système d'expérience.

### Fonctionnalités :
- Entitiées "Carte à gratter 1" et "Carte à gratter 2" (seule la carte à gratter "1" est fonctionnelle)
