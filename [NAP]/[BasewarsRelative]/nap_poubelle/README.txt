# Addon GMod - Poubelle Interractive

## Description

Cet addon pour Garry's Mod a été initialement conçu pour le mode de jeu **Basewars**, mais il est facilement adaptable à d'autres gamemodes tels que **DarkRP** ou tout autre système disposant d'une économie (argent) ou d'un système d'expérience.

### Fonctionnalités :
- Entité **"Poubelle"** pouvant être fouillée toutes les **120 secondes** par défaut.
- Chaque joueur possède un **cooldown personnel de 60 secondes** avant de pouvoir fouiller à nouveau une poubelle.
- Un **fichier de configuration** est fourni pour ajuster facilement les paramètres de l'addon.

## Description (English)

This Garry's Mod addon was originally created for the **Basewars** gamemode but can be easily adapted to other modes such as **DarkRP** or any gamemode using a money or XP system.

### Features:
- An **interactive "Trashcan" entity** that can be searched every **120 seconds** by default.
- Each player has a **personal cooldown of 60 seconds** before being able to search again.
- A **configuration file** is included to easily tweak the settings.

## 📁 Installation

1. Téléchargez les fichiers ou clonez le dépôt.

3. Modifiez le fichier de configuration selon vos besoins.
